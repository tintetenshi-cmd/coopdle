🎯 Problème Identifié

Problème : La sélection du nombre de lettres ne fonctionne pas lors de "Nouvelle partie"
Cause probable : L'élément DOM n'est pas accessible au moment du clic
Solution : Récupération dynamique de l'élément au lieu du cache

🔧 Solution Implémentée

Changements apportés :
✓ Récupération dynamique : document.getElementById("next-word-length")
✓ Logs de debug détaillés pour tracer le problème
✓ Vérification de l'élément au moment du clic
✓ Comparaison entre référence cachée et récupération dynamique
