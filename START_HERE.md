## 🚀 Déploiement Rapide (5 minutes)

### Étape 1 : Préparer Git
```bash
git init
git add .
git commit -m "Application Coopdle prête"
```

### Étape 2 : Pousser sur GitHub
1. Créez un nouveau dépôt sur [github.com](https://github.com/new)
2. Nommez-le `coop-wordle`
3. Puis :
```bash
git remote add origin https://github.com/VOTRE_USERNAME/coop-wordle.git
git branch -M main
git push -u origin main
```

### Étape 3 : Déployer sur Render
1. Allez sur [render.com](https://render.com)
2. Créez un compte (gratuit)
3. Cliquez sur **"New +"** → **"Web Service"**
4. Connectez votre dépôt GitHub `coop-wordle`
5. Render détecte automatiquement `render.yaml`
6. Cliquez sur **"Create Web Service"**
7. Attendez 2-3 minutes ⏱️

**🎉 C'est prêt !** Votre app sera sur : `https://coop-wordle-XXXX.onrender.com`
