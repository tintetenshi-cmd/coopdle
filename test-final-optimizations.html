🧪 Instructions de Test

Test sélection longueur :
1. Terminez une partie (gagnée ou perdue)
2. Choisissez une longueur spécifique (ex: 5 lettres)
3. Cliquez "Nouvelle partie"
4. Vérifiez que le nouveau mot a exactement 5 lettres
5. Le sélecteur doit revenir sur "Aléatoire"
Test idle game simplifié :
1. Vérifiez qu'il n'y a que 3 boutons : 💪 ⚡ 💡
2. Plus de boutons 🚀 (multiplicateur) et 🍀 (critique)
3. Stats affichées : Pièces, Par clic, Auto/sec, Total
4. Fonctionnalité simplifiée mais complète
Test messages de bienvenue :
1. Créez une partie coop
2. Faites rejoindre un autre joueur (ou ouvrez un autre onglet)
3. Vérifiez le message dans le chat avec effet feu 🔥
4. Format : "[Pseudo] a rejoint la partie ! [phrase aléatoire] 🔥"
