🧪 Instructions de Test

Test du bouton pied :
1. Allez dans l'idle game (colonne droite)
2. Vérifiez que le pied 🦶 est bien centré dans son cercle
3. Testez les effets de clic et hover
4. L'icône ne doit pas dépasser du cercle
Test des boutons upgrade :
1. Ajoutez des pièces : addCoins(10000)
2. Vérifiez que chaque bouton a une icône claire
3. Le coût est affiché en dessous de l'icône
4. Testez les achats et effets visuels
Test du sélecteur de longueur :
1. Terminez une partie (gagnée ou perdue)
2. Dans le résultat, choisissez une longueur spécifique
3. Cliquez "Nouvelle partie"
4. Vérifiez que le nouveau mot a la bonne longueur
Test des icônes aléatoires :
1. Allez dans la sélection de mode
2. Vérifiez les dropdowns Avatar, Couleur, Longueur
3. L'option "Aléatoire" doit avoir une icône ASCII
4. Pas d'emojis dans ces options
