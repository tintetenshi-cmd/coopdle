🧪 Instructions de Test

Test icônes uniformisées :
1. Allez dans la sélection de mode
2. Vérifiez que tous les dropdowns "Aléatoire" ont l'icône ⚡
3. Avatar, Couleur, et Longueur doivent être cohérents
Test idle game sans auto-farm :
1. Lancez une nouvelle partie
2. Vérifiez que "Auto/sec" affiche 0
3. Aucune pièce ne doit être générée automatiquement
4. Il faut acheter l'upgrade ⚡ pour avoir de l'auto-farm
Test nouvelle partie simplifiée :
1. Terminez une partie (gagnée ou perdue)
2. Vérifiez qu'il n'y a plus de sélecteur de longueur
3. Cliquez "Nouvelle partie"
4. La page doit se recharger complètement
5. Vous revenez à l'écran de titre avec tout réinitialisé
