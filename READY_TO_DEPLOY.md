## 🚀 Déploiement Rapide (3 étapes)

### 1. Initialiser Git (si pas déjà fait)

```bash
git init
git add .
git commit -m "Application Coopdle prête pour déploiement"
```

### 2. Pousser sur GitHub

```bash
# Créer un nouveau dépôt sur github.com
# Puis :
git remote add origin https://github.com/VOTRE_USERNAME/coop-wordle.git
git branch -M main
git push -u origin main
```

### 3. Déployer sur Render

1. Allez sur [render.com](https://render.com)
2. Cliquez sur "New +" → "Web Service"
3. Connectez votre dépôt GitHub
4. Render détecte automatiquement `render.yaml`
5. Cliquez sur "Create Web Service"
6. Attendez 2-3 minutes

**C'est tout ! 🎉**

Votre app sera disponible sur : `https://coop-wordle-XXXX.onrender.com`

## 🆘 Besoin d'Aide ?

### Problèmes courants

**L'app ne démarre pas**
- Vérifiez les logs sur le dashboard Render
- Vérifiez que `npm start` fonctionne en local

**WebSocket ne fonctionne pas**
- Render supporte WebSocket nativement ✅
- Vérifiez que vous utilisez bien Render/Railway/Heroku

**Erreur de build**
- Vérifiez que `npm install` fonctionne en local
- Vérifiez la version de Node.js (>=18)
